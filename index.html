Wprowadzenie

Dropwizard jest biblioteką/frameworkiem, który znacznie upraszcza tworzenie mikroserwisów, np. RESTful API.

Prosty w obsłudze ...

java -jar hello-world.jar server config.yml

... ale bogaty w środku (ficzery)

  1. Jetty (obsługa HTTP)
  2. JAX-RS (aka Jersey) (REST)
  3. Jackson (JSON)
  4. Metrics
  5. Guava
  6. Logback / slf4j
  7. Hibernate Validator
  8. Apache HttpClient, JerseyClient
  9. JodaTime
  10. Freemarker / Mustache (szablony)
